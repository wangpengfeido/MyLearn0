难度：困难

给定一个只包含 '(' 和 ')' 的字符串，超出最长的合法括号子串长度。

例一
````
Input: s = "(()"
Output: 2
Explanation: The longest valid parentheses substring is "()".
````

例二
````
Input: s = ")()())"
Output: 4
Explanation: The longest valid parentheses substring is "()()".
````

例三
````
Input: s = ""
Output: 0
````
