难度：中等

给定一个升序整数数组 nums，改定一个 target，找到 target 在 nums 中的开始结束位置。

如果 target 找不到，返回 [-1,-1]。

算法的时间复杂度最好为 O(log n)。

例一
````
Input: nums = [5,7,7,8,8,10], target = 8
Output: [3,4]
````

例二
````
Input: nums = [5,7,7,8,8,10], target = 6
Output: [-1,-1]
````

例三
````
Input: nums = [], target = 0
Output: [-1,-1]
````
