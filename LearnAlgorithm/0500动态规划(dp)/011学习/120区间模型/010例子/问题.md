## 问题
给定一个长度为n（n <= 1000）的字符串A，求插入最少多少个字符使得它变成一个回文串。

## 思路
假设d[i][j]表示字符串A[i,...,j]需要插入的字符串个数。

当 A[i]==A[j]，此时，
````
d[i][j] = d[i+1][j-1];
````

当 A[i]!=A[j],有两种情况：
1. 在A[j]后面添加一个字符A[i]
2. 在A[i]前面添加一个字符A[j]

所以，此时，
````
d[i][j] = min{ d[i+1][j], d[i][j-1] } + 1
````

边界条件是
````
当i>=j, d[i][j] = 0
````

注：时间复杂度为 O(n^2) ，并不是最优解法
